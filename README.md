### 框架设计思想
配置灵活但是繁琐
约定省事但是束缚
框架设计的初衷是为了简化我们的开发，我会再两者之间追求一个平衡

### 目录介绍

- **src** 源代码
    - **components** 组件库
        - ~~各自的组件库~~
            - 全局组件名称

关于组件的规范 
1.文件夹名称大驼峰命名
2.文件夹内文件统一为index.vue/ index.scss/ index.js
3.使用plop命令新建组件
4.**全局组件**：**大于等于两个**以上页面使用的模块需要抽析于components > componentsnamecomponentsname
